AI·딥러닝 검사 프로그램

불량 형태가 일정하지 않은 외관검사는 AI 모델과 기존 룰 기반 검사를 함께 구성하는 것이 효과적입니다. 학습 이미지 관리, 판정 기준, 신뢰도 표시, 오검 재검토 흐름까지 실제 운영을 고려해 설계합니다.

이력관리 프로그램

검사 결과, 바코드, 이미지, NG 사유, 작업 시간, 설비 번호를 저장하면 품질 문제 발생 시 원인을 추적하기 쉬워집니다. 제품별 레시피와 LOT 단위 이력관리를 함께 구성할 수 있습니다.

현장용 비전 프로그램에 필요한 기능

현장 작업자가 사용하는 비전 프로그램은 검사 알고리즘만 좋다고 충분하지 않습니다. 제품 변경 시 레시피를 쉽게 바꿀 수 있어야 하고, NG 발생 시 이미지를 바로 확인할 수 있어야 하며, 설비 신호가 끊겼을 때 원인을 알 수 있어야 합니다.

C#, Python, VisionPro, 딥러닝 모델, 바코드리더기 SDK, PLC 통신, 데이터베이스를 조합해 검사 목적에 맞는 전용 프로그램을 만들 수 있습니다. 기존 설비에 추가되는 경우에는 현재 PLC 주소, MES 통신 방식, 파일 저장 규칙을 먼저 확인합니다.

ASPEC은 검사 UI, 관리자 권한, 로그, 통계, CSV 저장, SQL 데이터베이스, 이미지 저장, 원격 유지보수까지 운영 관점에서 필요한 기능을 함께 검토합니다.
