Alignment

초정밀 비전 얼라인먼트

디스플레이 및 반도체 공정에서 필수적인 미세 정렬 솔루션입니다. 코그넥스 Patmax의 강력한 패턴 매칭 알고리즘을 활용하여 서브 픽셀 단위의 정밀도를 보장하며, 고속 로봇 가이던스를 수행합니다.

  • PatMax 알고리즘 적용
  • 로봇, 스테이지, 모터등의 통신/얼라인
  • 반복 정밀도 검증
Barcode Reading

고속 바코드/QR 리딩

물류 및 제조 라인의 이력 관리를 위한 고성능 바코드 판독 시스템입니다. DataMan 시리즈를 활용하여 손상되거나 인쇄 상태가 불량한 코드도 99.9% 이상의 인식률로 판독합니다.

  • 멀티 코드 동시 판독
  • DPM(Direct Part Mark) 난독 코드 대응
  • 고속 컨베이어 라인 적용
Inspection

정밀 외관 검사

제품의 품질을 보증하는 최종 관문인 외관 검사 시스템입니다. 고해상도 카메라와 특수 조명을 조합하여 스크래치, 찍힘, 이물 등 미세한 결함을 완벽하게 검출합니다.

  • 고해상도 카메라 적용 및 UI 개발
  • 다각도 조명 솔루션 제공
  • 실시간 불량 마킹 및 배출
Deep Learning

딥러닝 AI 검사

기존 룰 기반 알고리즘으로 해결하기 어려운 비정형 불량을 검출합니다. Cognex ViDi 및 Edge Learning을 활용하여 사람의 눈과 같은 감성 품질 검사를 자동화합니다.

  • 복잡한 배경의 결함 검출 (Red Tool)
  • 양품/불량품 분류 (Green Tool)
  • OCR/OCV 문자 인식 (Blue Tool)
Software

소프트웨어 제작

고객의 요구사항에 최적화된 전용 검사 소프트웨어를 개발합니다. C#, C++, Python 기반의 고성능 어플리케이션으로 직관적인 UI와 안정적인 장비 제어를 제공합니다.

  • 사용자 친화적 UI/UX 디자인
  • MES/POP 시스템 연동
  • 실시간 데이터 로깅 및 통계 분석
